# 라이즈 더 타워


## 게임 장르 : 3D 자동전투 등반 게임

## 게임 소개 : 장비를 뽑고 강화, 인챈트하여 탑의 최종층을 클리어하는 것이 목표인 게임입니다.


## 개발 목적 : 유니티 3D 게임 개발 능력 향상

## 사용 엔진 : UNITY 2021.3.25f1


## 개발 기간 : 약 12주 (2023.02.20 ~ 2023.05.15)
